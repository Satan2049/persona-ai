A supportive psychologist avatar with voice and lip-sync

Persona AI is a research demo: empathetic guidance in Persian and English, powered by a local LLM, optional FAQ-grounded RAG, and Piper text-to-speech — with a 2D avatar that speaks along.

Clone & run locally Installation guide

The interactive chat runs on your machine via the FastAPI backend — this page is the project home on GitHub Pages.

Built with

  • FastAPI backend + bundled web UI
  • OpenAI-compatible chat & embedding APIs
  • Piper TTS (piper1-gpl) — voices from Hugging Face
  • FAQ RAG over curated empathy prompts
  • Safety layer for high-risk content

Features

Designed for thesis demos and local experimentation — not a replacement for professional care.

Bilingual guidance

Locale-locked system prompts keep Persian and English replies consistent with the UI language.

Speaking avatar

Offline Piper voices, WAV playback, and viseme timelines drive lip-sync on a lightweight 2D face.

Grounded & safe

Optional RAG over an FAQ dataset plus escalation paths with configurable emergency numbers.

Run locally

GitHub Pages hosts this landing page only. To use the avatar, clone the repository, install dependencies, and start the backend on your computer.

1

Clone the repository

git clone https://github.com/USERNAME/persona-ai.git

2

Configure backend & Piper

Copy backend/.env.example to backend/.env, set your LLM API, install Piper, and download voice models into piper_models/. Full steps are in the README.

3

Start the server

On Windows, run backend\run.bat. Then open http://127.0.0.1:8000/ for the full avatar UI.
